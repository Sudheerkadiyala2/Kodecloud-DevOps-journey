# 🚀 DevOps & Cloud Hands-on Labs

## 📌 Overview

This repository contains hands-on labs and practical tasks focused on DevOps and Cloud technologies. The work is performed in real lab environments to build strong, job-ready skills.

---

## 🧠 Topics Covered

* Linux Administration (user management, permissions, processes)
* Networking fundamentals
* Shell scripting
* DevOps concepts
* Cloud fundamentals (AWS)
* Security and access control

---

## 📂 Repository Structure

The repository is organized based on concepts and tasks:

```
linux-user-management/
file-permissions/
ssh-access/
...
```

Each folder includes:

* `README.md` → explanation of the task
* Commands used
* Scripts (if applicable)

---

## ⚙️ Sample Task

### Create User with Expiry

```bash
sudo useradd -m -e 2027-03-28 rose
```

🔍 Verify:

```bash
sudo chage -l rose
```
